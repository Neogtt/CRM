# Masaüstü Kısayolu Oluşturuldu

Masaüstünüzde **"EXPO CRM.command"** adında bir kısayol oluşturuldu.

## Kullanım

1. Masaüstündeki **"EXPO CRM.command"** dosyasına çift tıklayın
2. Terminal penceresi açılacak ve uygulama otomatik olarak başlatılacak
3. Tarayıcı otomatik olarak http://localhost:3000 adresinde açılacak

## Özellikler

- ✅ Otomatik bağımlılık kontrolü
- ✅ Backend ve Frontend otomatik başlatma
- ✅ Renkli terminal çıktısı
- ✅ Hata kontrolü ve uyarı mesajları

## Notlar

- İlk çalıştırmada bağımlılıklar otomatik olarak yüklenecektir
- .env dosyası yoksa uyarı verilecektir
- Uygulamayı durdurmak için Terminal penceresinde Ctrl+C tuşlarına basın

## Sorun Giderme

Eğer kısayol çalışmazsa:

1. Terminal'i açın
2. Şu komutu çalıştırın:
   ```bash
   chmod +x ~/Desktop/EXPO\ CRM.command
   ```

3. Veya proje dizininden direkt çalıştırın:
   ```bash
   cd /Users/kemalcelikkalkan/localcrm
   ./start-crm.command
   ```

## Kısayolu Silme

Kısayolu silmek için masaüstündeki "EXPO CRM.command" dosyasını çöp kutusuna sürükleyin.
